# Career Index
Last updated: 2026-06-11

## Scaffold (empty until you use /career)

The `/career` skill writes to `Efforts/career-search/`:
- `tracker.md` -- application pipeline state
- `opportunities.md` -- scored active roles
- `applications/<company>-<date>/` -- application packets
- `scans/`, `retros/` -- portal scan results + retrospectives

Human-layer inputs you create:
- `Atlas/concepts/career/` -- goals, skills inventory, transition rules
- `private/resume.md` -- CV source of truth (gitignored)
- Career ref docs via `/deep` -> `Atlas/sources/career/`

Templates: `_templates/career-evaluation.md`.
