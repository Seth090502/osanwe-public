# Supplements / Health Index
Last updated: 2026-06-11

## Scaffold (empty until you use /health)

The `/health` skill writes to `Efforts/health-protocol/`:
- `current-stack.md` -- active supplement stack
- `bloodwork.md` -- lab history
- `peptide-protocol.md` -- active peptide protocol (if any)
- `analyses/`, `progress/` -- stack analyses + progress reports

Human-layer inputs you create:
- `private/health-baseline.md` + `private/medical-context.md` (gitignored;
  medical context is critical for correct lab interpretation)
- Supplement ref docs via `/deep` -> `Atlas/sources/supplements/`
  (evidence base, interactions, biomarkers, peptide protocols)
